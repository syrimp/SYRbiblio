HTML 기초

[< >] -> 태그라고 부름. 함수같은 기능들!
[li]는 목차 만들기.[ul](unordered list), [ol](ordered list). ul은 그냥 구분만 지어주고, ol은 목차에 순서매기며 구분지어줌.
[strong]은 글씨 진하게 만들어줌.
[u]는 underline, 밑줄 표시.
[h+숫자] -> 제목 의미. 숫자는 글씨크기, 숫자가 커질수록 글씨 작아짐. 숫자는 1~6까지.
[br] -> 줄바꿈표시. 닫는 태그[/br]같은게 필요 없음.

이건 단락을 바꾸는 기호인 [p]로, 뒤에 style="margin-top:45px;"를 써주면 여백(margin) 세밀하게 조절 가능.
다음은 이미지 파일 넣기.

[img]뒤에 src(소스코드)라는 속성값을 붙이고 주소 붙여넣기. 이미지 크기 조정은 [width] 사용.

[title]은 탭 창에 뜨는 홈페이지 이름 표시
[meta charset="utf-8"] - 유니코드의 인코딩 방식 중 하나로 이 형식으로 홈페이지를 열라고 브라우저에 알려주는 역할. 이거해야 한글 안 깨짐.
본문을 설명해주는, 실제로 보여지지는 않는 내용들이므로 [head]태그 사용하여 감싼다.
본문은 [body]로 감싸줌. 그리고 헤드와 바디를 [html]태그로 감싸줘야함.
또, 이 웹페이지가 HTML로서 만들어졌다는 것을 표현하기 위해서 문서의 시작에 추가.

링크를 나타내는 [a]태그- [a href="링크 걸 주소" target="_blank" title="링크 설명"] 뒤에 링크 걸 내용을 적고 [/a]로 마무리한다.
a는 anchor(닻)의 약자이고, href는 HyperText Reference의 약자.
target="_blank"는 링크를 클릭했을 때 새 창에서 페이지 열게하는 속성.
title은 링크가 어떤 내용인지 마우스 가져다대면 나오는 설명.
